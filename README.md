
# capacitor-ironsource

Capacitor plugin for native ironSource ad monetization.

## Install

To use npm

```bash
npm install capacitor-ironsource
```

To use yarn

```bash
yarn add capacitor-ironsource
```

Sync native files

```bash
npx cap sync
```
